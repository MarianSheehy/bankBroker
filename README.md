# Project Description
This project contains a bank brokering website which allows users to sign-up. They can create a catalogue of places in Ireland which they have spotted various banks


# Usage
- User can sign-up and log-in
- User can enter any number of places, the name of the bank spotted, the date and other interesting information to note
- User can upload an image relating to the place, or pictures taken of banks
- User details can be stored and deleted
- Places/banks can be stored and deleted
- The dashboard button brings the user back to the list of places


# Versions
Further versions will be issued, which allow places to be saved using a map and location data. There will also be a dropdown function implemented which allows the user to select from over 200 banks usually seen in Ireland. This database will expand and categorise the banks depending on which family the relate to
